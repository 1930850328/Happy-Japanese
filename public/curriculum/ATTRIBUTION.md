# Open curriculum attribution

The generated curriculum data in this directory uses the following open sources:

- Vocabulary: [open-anki-jlpt-decks](https://github.com/jamsinclair/open-anki-jlpt-decks), Copyright Jamie Sinclair, MIT License.
- Grammar and examples: [hanabira.org-japanese-content](https://github.com/tristcoil/hanabira.org-japanese-content), Creative Commons; attribution link required by the source project.
- Kanji readings and metadata: [kanjiapi.dev](https://kanjiapi.dev/) and KANJIDIC/EDRDG data, redistributed under the source data terms. JLPT level metadata originates from Jonathan Waller's JLPT resources.

The JLPT does not publish an official vocabulary, kanji, or grammar list for the current test. Level labels in these datasets are curriculum estimates, not official specifications.
